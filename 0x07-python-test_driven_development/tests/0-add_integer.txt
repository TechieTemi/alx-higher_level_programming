This module contain "add_integer()", which we are going to use to perform some operations

"add_integer" returns the sum of the argument passed to it using "+" operator.

It can work with an integer values.
>>> add_integer = __import__("0-add_integer").add_integer
>>> add_integer(1, 3)
4

It can also be use with a float value
>>> add_integer(4.0, 5)
9

The argument should not be more than 2
